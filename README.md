# Planète Sushi

A single-page, fully responsive and animated menu site for **Planète Sushi** —
a Japanese · Thai restaurant in Bordj Bou Arréridj, Algeria.

## Highlights
- 🐉 Hero built around the brand's red dragon (extracted to a clean transparent silhouette)
- ☯️ Perfect yin-yang light/dark theme toggle (remembers your choice)
- 🎬 Scroll-reveal animations, floating dragon, animated marquee, hover micro-interactions
- 📱 Mobile-first responsive layout, multi-column menu grid on larger screens
- ♿ Honours `prefers-reduced-motion`
- ⚡ Zero build step — a self-contained `index.html` + `dragon.png`

Palette: white · red · black, with subtle cream & gold accents.

Just open `index.html` in any browser.
